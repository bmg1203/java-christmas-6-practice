# 미션 - 크리스마스 프로모션

-----------------------------------------------------------------------
## 우아한 테크 코스 6기 프리코스 4주차 미션
- 사용 언어 : Java 17
- 개발 기간 : 23.11.10 - 23.11.15
- 개발자 : 김범창
- git : dk7648
- Folk : https://github.com/dk7648/java-christmas-6

## 사용자 입력 요구 사항

-----------------------------------------------------------------------
- 방문할 날짜는 1 이상 31 이하의 숫자로만 입력한다.


- (e.g 해산물파스타-2,레드와인-1,초코케이크-1)와 같은 메뉴 형식을 준수한다.
- 메뉴의 개수는 1 이상의 숫자만 입력한다.
- 중복 메뉴를 입력하지 않는다.

## 구현할 기능 목록

- [x] 메뉴 입력받는 기능
